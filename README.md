# A股主板10日短线选股器

这是一个本地运行的 A 股主板候选股筛选程序，目标不是每天硬凑 20 只，而是优先找“10 个交易日内更可能有短线弹性、但又没有明显前高压力”的候选。

新版核心思路：

- 只看主板非 ST 个股，不看创业板、科创板、北交所。
- 流通市值默认 25 亿到 350 亿，避开太小流动性差和太大拉不动的票。
- 先看走势，再看筹码；走势差的票不会因为散户减少或热度高被顶上来。
- 要求 20 日线、60 日线向上，且价格在 20 日线和 60 日线上方。
- 10 日涨幅默认 0% 到 14%，20 日涨幅 2% 到 22%，60 日涨幅 8% 到 40%，避免左侧弱票和短线过热票。
- 要求最近一段时间不能刚创新高：距 60 日高点至少 8 个交易日，距 120 日高点至少 12 个交易日。
- 离前高保留缓冲：离 60 日高点 5% 到 20%，离 120 日高点 6% 到 28%。
- 量能要温和放大，不能缩量没承接，也不能突然爆量过热。
- 股东户数半年至少更新 2 次，并把散户减少趋势作为强信号门槛。

主要输出：

- `outputs/a_share_stock_screener/report.html`
- `outputs/a_share_stock_screener/strong_signals.csv`
- `outputs/a_share_stock_screener/watchlist.csv`
- `outputs/a_share_stock_screener/stock_candidates.csv`

使用提醒：这是研究筛选器，不是自动交易系统，也不是买入指令。短线候选仍然需要结合当天市场环境、分时承接、止损位和仓位控制。
